⌚ Heart Rate Analyzer

📌 Overview
A mini project that analyzes workout heart-rate data to find the peak value, calculate stress index, and classify workout intensity.

⚙️ Technologies
Python
Tkinter (GUI)
Matplotlib

🧠 Working
Input: Heart-rate readings
Uses binary search to find peak heart rate
Calculates stress index (√peak)
Classifies workout intensity

🚀 How to Run
Run the Python file
Enter heart-rate values
Click Analyze and Show Graph

✨ Features
Peak detection
Stress analysis
GUI interface
Graph visualization

👨‍💻 Conclusion
Demonstrates efficient data analysis using algorithms and visualization for fitness monitoring.
